# Joystick

- https://create.arduino.cc/projecthub/MisterBotBreak/how-to-use-a-joystick-with-serial-monitor-1f04f0

### Using the joystick pins
- VCC
- GND
- VRx -> Variable resistance x
- VRY -> Variable resistance Y
- SW -> Switch / Button

### Connection
- MODULE -> ARDUINO
- VCC -> 5V
- GND -> GND
- VRx -> A0
- VRy -> A1
- SW -> Pin 2
